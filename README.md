## Running Locally

### Installation

1. Clone this repo

```bash
git clone git@github.com:marufmax/marufalom.com.git
```

2. Change directory

```sh
cd marufalom.com
```

3. Install yarn packages

```bash
yarn
```

1. Create a `.env.local` file following the `.env.example`

```bash
cp .env.example .env.local
```

6. Run the development server

```bash
yarn dev
```
